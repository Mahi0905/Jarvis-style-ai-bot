# 🌾 JarvisAI – FasalGuru Voice Assistant

JarvisAI is an AI-powered **vernacular voice assistant** designed to empower farmers with accessible agricultural intelligence. Built as part of the **FasalGuru project**, it bridges the digital literacy gap by providing real-time, localized farm advisory through natural voice interaction.

---

## 🚀 Features
- 🎙️ **Voice Interaction** – Farmers can ask questions in their local language.
- 🌱 **Agri-Advisory** – Crop recommendations, weather updates, and soil insights.
- 📊 **Dashboard Integration** – Backend powered by **FastAPI + PostgreSQL**, with a **React.js frontend** for monitoring.
- 🧠 **AI/ML Models** – Deployed via Hugging Face Spaces using **Gradio**.
- 🔗 **Scalable Architecture** – Designed for hackathon-ready deployment and nationwide adoption.

---

## 🛠️ Tech Stack
- **Frontend:** React.js
- **Backend:** FastAPI, Flask (JarvisAI microservice)
- **Database:** PostgreSQL
- **AI/ML:** Hugging Face Transformers, Gradio
- **Deployment:** Hugging Face Spaces, Docker
- **Voice Processing:** SpeechRecognition, gTTS
